# python_AV_car_simulation

## Introduction

This is a simulation of autonomous vehicle (AV) written in pygame. It is 2D to make it light and focuses on the algorithm implementation purpose onward

## Project Description

It was built from a sample code by https://tech.naja.io/blog/Simulating%20a%20Self-Driving%20Car%20in%20Python.html and currently being upgraded (1/4/2026). The project is developed in Visual Studio Code IDE, Ubuntu 20.04 operating system

## Prerequisite
- Python 3
- Python library: pygame, numpy, math
